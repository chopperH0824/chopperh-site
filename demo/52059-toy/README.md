# 52059 Toy Demo

入口文件：

[index.html](/Users/leyang862/Documents/copaw/demo/52059-toy/index.html)

使用方式：

1. 直接在浏览器打开 `index.html`。
2. 右上角 OFF / 静音 / 低 / 高 模拟实体拨杆，`OFF` 关机，其余三档开机。
3. 右侧“音频导入”支持一次导入整批 `mp3 / wav`，按文件名自动覆盖默认音频。
4. 长按逻辑在 demo 中统一缩短为 `0.8s`。

当前内置：

- 默认音频读取 `outputs/52059_v2_complete/run_20260421_155617/audio/`
- 音频需求映射读取你提供的 `52059_提示音需求表_v2.xlsx`
- 屏幕比例参考 `Math toy display design Mladen.pdf`

实现说明：

- `找最大 / 最小` 已改成参考模拟器规则：一位数题直接按屏幕上的答案数字，两位数题输入完整答案后按 `OK` 提交。
- `猜规律` 题型已收敛到参考模拟器的短序列题：等差、交替、跳数、倍增，答案填在下一空位。
- 计算模式与比大小模式已接入 v6 PRD 的算式拼接播报：`0-20` 直接播，`21-99` 按“整十 + 个位”拼接；计算模式追加 `Equals?`，比大小模式只播算式部分。
- 数字拼接语音素材在需求表里是打包写法，demo 内部映射到了拆开的 `VO_num_* / VO_op_*` 文件。
- 音乐模式现在按 v6 规则处理翻页音：手动按 `< / >` 播放 `SFX_music_page`，自动翻页保持静默。
- `关机保存进度` 与 `每次上电恢复初始状态` 在 PRD 中互相冲突，demo 当前优先实现“关机 / 休眠后恢复会话”，`冷启动` 按钮用于强制回到初始状态。

调试参数：
